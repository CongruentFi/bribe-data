# Congruent Bribe Data Generator

Bribe Data Generator for Congruent.

The Airdrop will happen on Polygon.

## Usage

To generate merkle tree for claiming:

```shell
$ yarn
$ yarn ts-node src/gen-tree.ts <day>
```
